## ¿Como usar la aplicación de forma local? <a name="4"></a>

Este proyecto requiere **[Node.js](https://nodejs.org/)** para su ejecución.

```
git clone https://github.com/Agushtinsin/digigames
```
```
cd digigames
```
```
npm install
```
```
npm start
```
